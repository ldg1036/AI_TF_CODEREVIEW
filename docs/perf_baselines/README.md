# Baseline Files

Last Updated: 2026-02-27 (real UI/HTTP baselines collected)

This directory stores saved performance baselines used for regression checks.

## File Types

- `http_perf_baseline_*.json`
  - Generated by `tools/http_perf_baseline.py`
  - Contains `/api/analyze` metrics + HTTP wall-clock times

- `ui_benchmark_baseline_*.json`
  - Generated by `tools/playwright_ui_benchmark.js`
  - Contains frontend rendering/scroll/jump timings

- `ui_thresholds_*.json`
  - Recommended threshold values used in CI/local checks

## Notes

- Keep baselines grouped by machine/profile when possible.
- Prefer multiple iterations and compare `p95`, not single runs.
- Record environment metadata (CPU/RAM/Python/Node/Ctrlpp/LLM provider).
